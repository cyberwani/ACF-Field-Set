== ACF Field that lets you add sub fields like a repeater. ==

ACF Field that lets you add sub fields like a repeater.  Does not repeat, but let's you create groups of fields with sub field requirements, while allowing the field set as a whole to be not required.

= Compatibility =

This ACF field type is compatible with:
* ACF 5

== Installation ==

1. Copy the `acf-field_set` folder into your `wp-content/plugins` folder
2. Activate the Field Set plugin via the plugins admin page
3. Create a new field via ACF and select the Field Set type
4. Please refer to the ACF Field that lets you add sub fields like a repeater. for more info regarding the field type settings
